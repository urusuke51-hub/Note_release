# Phase 0 バックテストツール（コスト実装版）— 使い方ガイド

note 記事「7年検証したら分かった。「強い戦略」は存在しなかった」のダウンロード特典です。

---

## 同梱ファイル

| ファイル | 内容 |
|----------|------|
| `backtest_portfolio.exe` | ポートフォリオバックテスト実行ツール |
| `report_html.exe` | インタラクティブHTMLチャート生成ツール |
| `README.md` | このファイル |

---

## 動作環境

- **OS**: Windows 10 / 11（64bit）
- **Python**: 不要（インストール済み不要）
- **インターネット接続**: 必須（株価データを yfinance から取得します）

---

## 使い方

### 1. バックテストを実行する（backtest_portfolio.exe）

exe ファイルをダブルクリック、またはコマンドプロンプトで以下を実行します。

```
backtest_portfolio.exe
```

**よく使うオプション**

```
backtest_portfolio.exe --cash 500000          # 初期資金を50万円に変更
backtest_portfolio.exe --max-pos 3            # 最大同時保有を3銘柄に変更
backtest_portfolio.exe --with-fundamentals    # 業績フィルター付きで実行（推奨）
backtest_portfolio.exe --years 5              # 直近5年間で検証
backtest_portfolio.exe --start 2019-01-01     # 2019年以降のみエントリー
backtest_portfolio.exe --loose-po             # 長期検証モード（コロナ前後対応）
backtest_portfolio.exe --loose-po --start 2019-01-01   # 記事の7年検証を再現
```

実行後、`results/` フォルダに以下が生成されます：
- `portfolio_trades.csv` — トレードログ一覧

---

### 2. チャートを生成する（report_html.exe）

バックテスト後、個別銘柄のトレードを目視確認できるHTMLチャートを生成します。

```
report_html.exe --code 7203
```

証券コードは4桁の数字で指定します（例：7203=トヨタ、6758=ソニー）。

実行後、`results/` フォルダに `7203_chart.html` が生成されます。ブラウザで開いてください。
- ドラッグ：横スクロール
- ホイール：ズームイン・アウト
- ローソク足にカーソル：OHLCV・トレード詳細を表示

---

## 注意事項

- **実行タイミングによって結果が変わります**。yfinance が取得する株価は配当・株式分割を都度調整するため、記事掲載時の結果と完全一致しない場合があります。
- **初回実行は時間がかかります**。全上場銘柄（約1,350銘柄）の株価データをダウンロードするため、10〜30分程度かかります。2回目以降はキャッシュを利用するため高速です。
- **ウイルス対策ソフトの誤検知について**。PyInstaller でビルドした exe は Windows Defender に誤検知されることがあります。その場合は除外設定を追加してください。
